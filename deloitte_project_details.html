Deloitte Data Analytics Job Simulation

Transforming manufacturing data into strategic insights for operational efficiency

June 2025
Excel, Tableau
Manufacturing Analytics

Project Overview

This Deloitte Data Analytics Virtual Job Simulation provided hands-on experience with real-world data challenges faced by analysts in a professional services environment. The project involved analyzing manufacturing data from Daikibo Industries to identify operational inefficiencies and gender pay equity issues across global factories.

Key Objectives

  • Analyze machine health data to identify maintenance priorities
  • Develop a gender pay equity assessment algorithm
  • Create actionable visualizations for executive decision-making
  • Provide strategic recommendations based on data insights

Machine Health Analysis

Problem Statement

Daikibo Industries needed to optimize maintenance processes by identifying patterns in machine failures across their global manufacturing facilities. The analysis focused on classifying machine health status and identifying factories with the most critical needs.

Data Analysis Approach

Using Excel, I performed data classification and analysis to:

  • Categorize machines by health status (healthy, needs repair, unhealthy)
  • Quantify failure rates by machine type and location
  • Identify patterns in failure frequencies

Data Visualization

The findings were presented in an interactive Tableau dashboard that enabled stakeholders to:

  • Compare machine health across locations
  • Identify most problematic machine types
  • Track maintenance priorities visually

Interactive Tableau dashboard visualizing machine health across factories

Key Findings: Machine Health

  • Seiki Factory: Critical situation with 480 unhealthy machines (all Laser Welders)
  • Shenzhen Factory: 420 unhealthy machines across multiple machine types
  • Meiyo Factory: 110 unhealthy machines (70 Heavy-duty drills & 40 Laser cutters)
  • Berlin Factory: 20 unhealthy machines (all furnaces)

Strategic Recommendations

  • Immediate predictive maintenance program for Seiki's Laser Welders
  • Comprehensive maintenance review for Shenzhen's diverse machine issues
  • Standardized repair schedules for Meiyo's high-failure machines

Gender Pay Equity Analysis

Problem Statement

The company needed to assess potential gender pay disparities across all job roles and locations following internal complaints about compensation fairness.

Analytical Approach

Developed a pay equity algorithm in Excel to:

  • Quantify pay disparity levels across all job roles
  • Evaluate equality metrics company-wide
  • Identify locations with the most significant disparities

Excel-based pay equity analysis showing compensation comparisons

Key Findings: Pay Equity

  • Meiyo Factory: "Highly Discriminative" classification with severe disparities
  • Seiko Factory: Moderate pay disparities requiring attention
  • Berlin & Shenzhen: Most equitable compensation structures

Strategic Recommendations

  • Immediate pay audit for Meiyo's highest-disparity roles
  • Development of corrective action plans for Seiko
  • Adoption of Berlin/Shenzhen best practices company-wide

Project Impact

This simulation provided practical experience in transforming raw data into strategic business insights. Key learnings included:

  • Importance of data cleaning and preparation in the analysis process
  • Effective visualization techniques for different stakeholder groups
  • Translating technical findings into actionable business recommendations
  • Balancing quantitative analysis with qualitative context
